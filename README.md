PARA ARRANCAR EL PROYECTO LA PRIMERA VEZ

git clone (repositorio)

entrar en la carpeta del repositorio con el programar para trabajar con el código y escribir en la consolael siguiente comando para instalar
las dependencias (las librerías que esta usando)

npm install

Tenemos que copiar y pegar .env-example y lo renombramos como .env y rellenamos las variables de entorno que hay dentro

Y arrancamos lanzando este comando

npm run dev
